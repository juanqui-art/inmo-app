# Concise Mode

Activates **Concise Mode** - provides direct, brief responses.

**Use this when you want:**
- Brief, straight-to-the-point answers
- No unnecessary explanations
- Quick responses without context
- Minimal elaboration

## What Changes

Claude switches to concise mode:
- Removes verbose explanations
- Gets to the point immediately
- Skips background information
- Focuses only on what you asked

## Example Usage

```
/conciso What's the issue with this code?
/conciso How do I fix the TypeScript error?
/conciso Why isn't this working?
```
