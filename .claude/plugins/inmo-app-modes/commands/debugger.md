# Debugger Mode

Activates **Debugger Mode** - a systematic problem-solving specialist.

**Use this when you want:**
- Root cause analysis
- Scientific debugging methodology
- Systematic problem investigation
- Multiple hypotheses tested

## What Changes

Claude switches to debugging specialist mode:
- Identifies root causes methodically
- Proposes multiple testable hypotheses
- Provides specific debugging steps
- Uses browser/IDE tools effectively
- Explains WHY problems occur

## Example Usage

```
/debugger Why is the map re-rendering every 100ms?
/debugger Investigate this API timeout issue
/debugger Find the memory leak in the property gallery
```

---

This command invokes the `debugger` agent with system prompts optimized for systematic problem-solving and diagnosis.
